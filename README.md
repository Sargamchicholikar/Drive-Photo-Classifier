# Drive Photo Classifier

Drive Photo Classifier is a Google Chrome extension that automatically 
organises your Google Drive photos into folders — Animals, People, 
Groups, and Junk — using an on-device AI model.

Your photos are never sent to any third-party server.
Photo classification happens entirely inside your browser.
